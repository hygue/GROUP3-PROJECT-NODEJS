 Built With (Core Technologies)

As required by the student attendance project, the system is developed using:
Runtime Environment: Node.js
Backend Framework:Express.js
Database Management:MySQL


 Key Features & API Functionalities

The system allows users to perform the following operations:

Add Students: Create new student records.
  Record Attendance: Mark and save daily attendance.
  View Attendance Records: Retrieve attendance history.
  Update Student Information:Modify existing student details.
  Delete Student Records:Remove students from the system.
